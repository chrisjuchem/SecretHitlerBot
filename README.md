## Setup

[Install nvm (Node Version Manager) here.]( https://github.com/creationix/nvm)

run  `nvm install 11.4.0`

run  `npm install`

start the bot:  `npm start` 

## .env

create a .env file with these keys and the appropriate values for your bot.

```
CHAT_BOT_ID="abc"
PM_BOT_ID="123"
CHAT_BOT_PORT=5858
PM_BOT_PORT=5859
```
